Jak stworzyć stronę internetową

Czym jest HTML?

HTML (HyperText Markup Language) to podstawowy język używany do tworzenia stron internetowych. Umożliwia strukturalne przedstawienie treści, w tym tekstów, obrazów, linków i innych elementów multimedialnych. Każda strona internetowa jest zbudowana z elementów HTML, które przeglądarka internetowa interpretuje i wyświetla użytkownikowi. HTML jest fundamentem sieci WWW, a jego znajomość jest kluczowa dla każdego, kto chce tworzyć lub edytować strony internetowe.

Podstawowe znaczniki HTML

  • <!DOCTYPE>

    Definiuje typ dokumentu i wersję HTML, której używamy.

  • <html>

    Najwyższy element w dokumencie HTML, zawiera wszystkie inne elementy.

  • <head>

    Zawiera metadane dokumentu, takie jak tytuł i linki do stylów CSS.

  • <title>

    Definiuje tytuł dokumentu wyświetlany na karcie przeglądarki.

  • <meta>

    Przechowuje metainformacje o stronie, takie jak kodowanie znaków i opis.

  • <body>

    Zawiera treść strony, w tym teksty, obrazy, linki i inne elementy.

  • <h1> do <h6>

    Definiują nagłówki różnego poziomu, od najważniejszego (<h1>) do najmniej ważnego (<h6>).

  • <p>

    Definiuje paragraf tekstu.

  • <a>

    Tworzy hiperłącze do innego dokumentu lub zasobu.

  • <img>

    Wstawia obraz na stronie.

  • <ul>

    Tworzy listę nienumerowaną (punktowaną).

  • <li>

    Definiuje pojedynczy element listy.

  • <div>

    Służy do grupowania i stylizowania sekcji dokumentu.

  • <span>

    Używany do stylizowania małych fragmentów tekstu.

  • <br>

    Wstawia pojedynczy odstęp (przełamanie linii).

Czym jest CSS?

CSS (Cascading Style Sheets) to język służący do opisywania wyglądu i formatowania strony internetowej stworzonej w HTML. CSS umożliwia oddzielenie struktury dokumentu od jego prezentacji, co ułatwia zarządzanie wyglądem strony. Stylowanie przy użyciu CSS obejmuje zmiany w kolorach, czcionkach, odstępach, rozmieszczeniu elementów i wiele innych aspektów wizualnych.

Podstawowe reguły CSS

  • color

    Określa kolor tekstu elementu.

  • background-color

    Ustawia kolor tła elementu.

  • font-size

    Definiuje rozmiar czcionki tekstu.

  • font-family

    Określa rodzaj czcionki używanej w tekście.

  • text-align

    Wyrównuje tekst wewnątrz elementu (np. do lewej, prawej, wyśrodkowany).

  • margin

    Określa zewnętrzne odstępy wokół elementu.

  • padding

    Określa wewnętrzne odstępy wewnątrz elementu.

  • border

    Definiuje obramowanie elementu, jego szerokość, styl i kolor.

  • width

    Określa szerokość elementu.

  • height

    Określa wysokość elementu.

  • opacity

    Ustawia przezroczystość elementu.
